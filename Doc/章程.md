#P2俱乐部章程

####一、总则
P2俱乐部是研究和制订利益共同体规律、方法学和工具的俱乐部，并为会员部署利益共同体提供支持。

####二、角色清单
1. 会员: 应在每年初始期前,自行指定六个月的活跃时间，活跃时间内每个月至少完成10个贡献值（即为最低贡献值）。
2. 高级会员: 应在每年初始期前,自行指定八个月的活跃时间，活跃时间内每个月至少完成100个贡献值（即为最低贡献值），并在活跃时间内按照管理员要求值班。
3. 管理员：负责俱乐部的管理，制定章程以外的规章及表单。

1. 资金
2. 固定资产

        4. 案例  
    4. 俱乐部运营计划（含预算计划）（2）
    5. 管理员工作报告（含决算报告）（3）
    6. 管理制度（含部门设置、报酬方案、各类信息传递方式）（2）
    7. 会员名录（含会员的角色、贡献值、现金、可提供的服务等）（5）
    8. 行权通告（5）
    9. 资料征集目录（含贡献值、审议标准）（3）

####五、协作过程
1. 会员管理
	1. 会员加入
		1. 认可本章程，提出申请，缴纳会费，即可成为会员、高级会员。
		2. 申请者提交申请，具体内容和方式由管理员制定并公布。
		3. 管理员检查内容符合要求后，通知申请者缴纳会费。
		4. 管理员办理**收款**后，更新《会员名录》。
		5. 申请者成为会员或高级会员，使用其申请的新会员ID。
	2. 会员退出
		1. 会员、高级会员提出申请，即可退出俱乐部。
		2. 会员、高级会员提出申请，具体内容和方式由管理员制定并公布。
		3. 管理员检查内容符合要求后，更新《会员名录》并通知申请者。
		4. 退出后，会员、高级会员已获得的贡献值清零，会员ID永久不再使用。
	3. 会员信息修改
		1. 会员、高级会员提出申请，具体内容和方式由管理员制定并公布。
		2. 管理员检查内容符合要求后，更新《会员名录》并通知申请者。
	4. 辅导
		1. 出现以下情况的会员、高级会员，由管理员更新《会员名录》清零贡献值，中止会员资格；发出《行权通告》指派一名高级会员作为其导师，指导其完成相应责任。
			1. 未能完成最低贡献值；（辅导期六个月）
			2. 活跃时间未公布实践情况；（辅导期三个月）
			3. 活跃时间未针对工作难点提交会员见解。（辅导期三个月）
		2. 辅导期结束后由导师向会员大会提交《审议记录》汇报其情况，由会员大会表决永久劝退或恢复原角色。
		3. 指导期间，被指导的会员应根据实际工作内容向导师支付报酬，价格按非会员服务价格的120%计算。

2. 会员交流
	1. 在线讨论
		1. 具体讨论方式由管理员决定并公布，相关账号密码由管理员维护。
		2. 活跃时间内的会员、高级会员应每旬公布一轮实践情况，内容包括：
			- 所在单位的相关实践情况；
			- 所遇到的问题及现实中采取的解决方案。
	2. 提炼重点问题：由值班高级会员在每月初从各会员所在单位、俱乐部的工作中提炼一个工作难点。
	3. 提交会员见解
		1. 活跃时间内的会员、高级会员应针对当月工作难点提交会员见解，讲解解决方案，有条件的应安排实践自检。
		2. 值班高级会员在月末应汇总各会员见解。
		3. 会员见解对俱乐部以外公布，如果整理为资料库投稿可以按照相应规则获得贡献值。
	4. 召集会员大会
		1. 管理员可以召集会员大会。
		2. 三分之一以上的会员、高级会员联名可以召集会员大会。
		3. 召集人应提前一周通知各会员、高级会员，并附上详细议程。
	5. 出席会员大会
		1. 召集方式、议程符合要求的，会员、高级会员可以出席会员大会。
		2. 超过三分之二的会员、高级会员出席的，会员大会有效。
		3. 以下事项以贡献值为权重表决，得票超过全体会员、高级会员贡献值总数的三分之二为通过：
			- 修改章程
			- 选举管理员
			- 审议《管理员工作报告》
			- 更新《俱乐部运营计划》
		4. 其它事项按一人一票表决，得票超过全体会员、高级会员人数的二分之一为通过。
		5. 议事规则的其它内容由管理员制定并公布。
	6. 签名：会员应使用会员ID对应的公钥进行数字签名并在当天提交。

3. 资料库管理
	1. 资料征集：管理员制定并提交《资料征集目录》。
	2. 专案征集
	3. 投稿
		1. 会员、高级会员按照《资料征集目录》要求，均可提交资料，并要注明参与作者的信息、第一作者和贡献值分配规则，选择是否参与贡献值预分配。
		3. 管理员汇编资料
	4. 预分配报名
	5. 预分配审议

**3.资料征集：**

(1) 总编辑应按照要求，定期制定《资料征集目录》。

(2) 会员、高级会员按照《资料征集目录》要求，均可提交资料,并要注明参与作者的信息、第一作者和贡献值分配规则。

(3) 管理员汇编资料，资料第一作者可以代表全体参与作者，选择是否参与贡献值预分配。如参与，则按照预分配审议规则进行评审，确定和发放预分配贡献值，最后再参加封闭交流审议，确定和发放最终贡献值。如不参与，则提交最近一次封闭交流审议，确定和发放最终贡献值。

(4) 总编辑根据审议结果，整理资料库，发布新《资料清单》。

(5) 管理员根据审议结果，调整《会员名录》中相应会员的贡献值。

**4.专案管理**

(1)高级会员每年应指定八个月的服务时间，由管理员编制值班表。

(2)专案提出有2种方式：

I. 会员、高级会员均可提交专案申请；

II.总编辑可根据资料库中的案例提出专案申请。

(3)会员、高级会员或总编辑提出的专案，应明确以下内容：

I.案例类型：COM设计或COD部署。

II.问题范围：规律、方法学或工具。

(4) 管理员编写《专案征集通知》发给值班高级会员。

(5) 除非当月已完成100个贡献值，否则值班高级会员必须在指定时间内提交对应的专题报告，其他会员、高级会员也可自愿提交。任何会员提交专题报告时，均要注明参与作者的信息、第一作者和贡献值分配规则。

(6)根据专案提出方式，按照以下根据处理贡献值：

I.由会员、高级会员提出的专案申请：专题报告作者可选择是否参与贡献值预分配，如参与，则按照预分配审议规则进行评审，确定和发放预分配贡献值，扣减提出专案申请的会员或高级会员相应的贡献值，最后再参加封闭交流审议，确定和发放最终贡献值，再扣减提出专案申请的会员或高级会员相应的贡献值。如不参与预分配，则提交最近一次的封闭交流审议，确定和发放最终贡献值，再扣减提出专案申请的会员或高级会员相应的贡献值。

II.由总编辑提出的专案申请：专题报告作者可选择是否参与贡献值预分配，如参与，则按照预分配审议规则进行评审，确定和发放预分配贡献值，最后再参加封闭交流审议，确定和发放最终贡献值。如不参与预分配，则提交最近一次的封闭交流审议，确定和发放最终贡献值。

(7)总编辑根据审议结果整理资料库，更新《资料清单》。

(8)管理员根据审议结果，调整《会员名录》中相应会员的贡献值。

**5.封闭交流活动**

(1) 管理员制定的《俱乐部运营计划》、《预算计划和决算报告》中应明确封闭交流的报名时间、活动时间、地点、预算等，明确参与者所需贡献值。

(2)会员、高级会员自愿报名参与封闭交流活动，高级会员应每年至少参加一次。名额有限时，根据报名时间先后顺序确定参会人员。

(3)管理员安排封闭交流的后勤工作，如预算不足，应制定参与者分担方案。默认方案是根据贡献值高低，报销参与者的费用，未报销的自费。

(4)参与者以贡献值为权重进行投票，选出一名协调人。

(5)协调人制定议程并指定总结人。议程应包括：
    
I. 讨论提炼近期俱乐部研究领域出现的重点、难点、疑点问题；

II. 针对重点、难点、疑点问题，制定解决方案。

(6)总结人主持所负责内容的讨论，并整理为信息资产。

(7)协调人主持和审议管理员和总结人提交的内容，根据《资料征集目录》中的审议标准评定等级，提交《审议记录》。

**6.宣传及营业**

(1)宣传工作由管理员在《部门设置、报酬方案和管理制度》中明确。

(2)非会员购买俱乐部服务时，如果有明确的引入会员，部分税后收入分配给引入会员。

**7.人事管理**

(1)管理员候选人在选举前应该提交《俱乐部运营计划》和《预算计划和决算报告》，通过召开会员大会，由会员以贡献值为权重选举产生，得票多者当选。

(2)管理员应每年度的第一个月提交上年度《管理员工作报告》，并更新《俱乐部运营计划》和《预算计划和决算报告》，由会员大会以贡献值为权重表决，半数通过则继续任职。否则，由除管理员外，贡献值最高的会员召集在第二个月末之前改选，当选新管理员立即赴任。

(3)总编辑候选人在选举前应该提交《资料征集目录》，由会员大会，按照按一人一票选举产生。

(4)司库由会员大会按一人一票选举产生，司库根据《行权通告》调动资金和固定资产。发言人由管理员指定。

(5)监事应定期检查俱乐部财务情况，监督管理员、总编辑、司库、发言人、会员等角色的职务行为，若与俱乐部章程或制度不符合的，由监事召集会员大会处理。

(6)每个角色竞选时，要递交下一个财务年度的工作内容和目标，经会员大会表决通过后作为角色阶段性考核指标。每个角色在一个财务年度中，如果任何考核指标未按要求完成，由会员大会重新选举该角色。

(7)每个角色竞选时，以票数最多且超过三分之二者为胜。如参选者均未达到三分之二，则去掉最低得票者重新投票，直至选出票数最多且超过三分之二者为胜。

(8)俱乐部任何人员如有违反章程行为，任何会员可提议召开会员大会，提议经三分之二会员支持，即可召开会员大会进行处罚。处罚决定由参会的三分之二会员同意，即为通过。

(9)发言人定期收集俱乐部信息对外发布，同时接受外部媒体采访。除发言人之外，俱乐部其它会员不得对外发布俱乐部的任何信息。发言人应每年度的第一个月提交上年度的《宣传清单》。

(10)管理员、总编辑、司库、发言人、监事任期不受限制。

**8.日常管理**

(1)等级小于等于3的非资料库信息资产，经会员大会审议后生效。

(2)信息资产对会员公开。

(3)会员对外提供服务，如需更新客户信息到俱乐部的会员名录，俱乐部收取一定比例的管理费，同时俱乐部记录整个服务过程的主要信息，为后续外部客户提供参考。

(4)根据俱乐部账务情况，司库定期更新《出纳账本》，管理员定期更新《会计账本》。监事至少每2个月进行一次财务检查，主要检查《出纳账本》、《会计账本》、俱乐部现金余额和银行存款是否一致，并检查资金使用是否合理和合规，并签名确认。

(5)俱乐部定期召开线上讨论会议，由管理员提前汇总议题并召集，按照俱乐部议事规则进行。

(6)本章程修改时，由会员大会讨论表决，会员以贡献值为权重进行投票，得票超过三分之二为通过，否则不通过。

####六、利益分配
1.会员的贡献值可以兑换设计COM和部署COD的规律、方法学和工具等的智力支持。

2.预分配审议规则：

(1)如果当月有要求预分配的资料或专案，由会员、高级会员自愿组成不超过三人的预分组，多人报名时按先后顺序选择分配组。参与预分组的会员、高级会员在封闭交流时不再参与这些内容的审议。总编辑每年至少参与八次预分配，不受报名顺序限制。

(2)预分组可以为申请预分配的信息资产分配贡献值，并提出理由。管理员将贡献值立即拨付作者，并把预分配结果一并提交封闭交流。

(3)如果封闭交流中，该信息资源获得的贡献值超过预分配的贡献值，只补发超出部分，该信息资源获得的预分配贡献值的20%作为奖励，发给预分组成员。如果封闭交流分配的贡献值低于预分配的贡献值，以差额部分作为处罚,由预分组成员上交。

(4)奖惩分配：总编辑与每名其他预分组成员按1.3:1权重加权分配奖惩。
